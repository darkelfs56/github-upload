Monitor is the display used to do work on a PC.
Key Features:

  • Resolution - this refers to the amount of pixels in the display. In general, bigger number equals better display. The most common resolution is Full HD or 1920 x 1080 for general use, however there are bigger resolution such as 4k and above. It is important to note that when choosing a resolution, the processors can handle the workload.
  • Refresh rate - this refers to how often the display updates the screen. For general use 60 Hz is enough. For gaming, 144 Hz is recommended.
  • Size - this refers to the size of the display measured diagonally. For general use 22-32 inch is enough. For gaming, it is up to your budget and the type of game you want to play as it can get up to 50 inch or more
  • Type - this refers to the type of panel the display uses (TN/VA/IPS). TN monitors are the fastest but cheapest, due to poorer image quality when viewing from a side angle. IPS monitors have slightly faster response times and show color better than VA panels, but VA monitors have the best contrast out of all three panel types.

Case is the case that holds all the PC components inside it.
Key Features:

  • Size - this refers to the size of the case. There are three case sizes: full tower, mid-tower and mini-ITX. When choosing a case, it is important that you know the size of the motherboard so that it fits.
  • Amount of fan - this refers to how many fans can be fitted into the case.
  • Usb options - this refers to what type of usb options it offers

Mouse is a device used to navigate a computer screen.
Key Features:

  • DPI - this refers to the sensitivity of the mouse in dots per inch (DPI). Higher DPI means that less physical movement is required to move the cursor across the screen.
  • Ergonomics - this refers to the design of the mouse and how it affects our well-being.
  • Connection - this refers to what type of connection used by the mouse. The connection of a mouse can be wired or wireless.
  • Buttons - this refers to the amount buttons that a mouse have and whether they are programmable or not.

Keyboard is an input device used to enter alphabets, number, symbols and other commands.
Key Features:

  • Switch type - this refers to the type of mechanism used in the keys of the keyboard. The three common types are rubber dome switch, scissor switch and Cherry MX switches. Rubber dome switch is popular in cheap keyboards. Scissor switch is normally used in laptop keyboards. Cherry MX switches are mainly used for gaming and there are a lot of varieties available. For more info on Cherry MX switches and its varieties pls go to this link.
  • Size - this refers to the design and the size of the case. Common ones are 100% or a full size keyboard with numpad and 60% which is the size of keyboard on most laptops.
  • RGB - this refers to whether RGB lighting is available on the keyboard or not.
  • Connection - this refers the connection type, whether it is wired or wireless.
